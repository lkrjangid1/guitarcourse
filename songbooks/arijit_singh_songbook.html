How to use this songbook

Capo

Most songs use a capo so you can play with simple G, C, D, Em shapes. Match the capo position shown.

Strumming

Each song lists its strum pattern. D = down, U = up, _ = skip

Color-coded chords

Same chord = same color across every song. Helps you spot patterns instantly.

Tempo

Start at half the listed tempo. Speed up only when changes are clean.

Search

Use the search box at the top to filter by song name or movie.

Links

Every song has YouTube tutorial, official audio, and Spotify links — tap to listen.
